# Magnetic hexaedral blocks Self-Consistent Solver

This MATLAB script computes the magnetic response and force interactions of a set of magnetizable hexahedral blocks subjected to an external magnetic field generated by a circular coil source.

The code performs a self-consistent iterative solution to determine the magnetization of each block, taking into account:

- the external magnetic field produced by the coil;
- the magnetic field generated by the other block;
- the self-field contribution of each hexahedral blocks;
- the nonlinear magnetic material law of the fixator;
- the resulting dipole-dipole and Kelvin force contributions.

## Main Features

The script:

1. Defines the geometry of three hexahedral blocks.
2. Computes the external magnetic field along the coil axis.
3. Initializes the magnetization of each block.
4. Iteratively updates the magnetization until convergence.
5. Computes magnetic-field gradients using finite differences.
6. Evaluates force contributions using:
   - a magnetic dipole force formula;
   - Kelvin force formulation;
   - external source contribution.
7. Plots the convergence error.
8. Visualizes each hexahedron with its face normals.
